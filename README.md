# Hotel-Booking-Cancellation-Prediction

# 🚀 From 119,000 Raw Records to a 92% Accurate Prediction System

💥 A real-world Data Science & Business Intelligence project solving one of the biggest challenges in the hospitality industry:

👉 Why do hotel customers cancel their bookings?

📊 Using advanced data analysis, machine learning, and database design, this project uncovers hidden patterns, predicts cancellations, and provides actionable strategies to reduce revenue loss.

⚠️ At one point, the model achieved 100% accuracy…
But that turned out to be a critical mistake (data leakage).

✅ After fixing it, the model reached a realistic and powerful 92.7% accuracy.

💡 This project is not just about data — it's about making better business decisions.
